What is a METAR?

A METAR (Meteorological Aerodrome Report) is a standardized weather observation issued by airports around the world, typically every hour or whenever conditions change significantly. Pilots use METARs to assess weather before and during a flight, but the format was designed for efficiency, not readability — making it look like a string of cryptic codes to everyone else.

Here is what a real METAR looks like:

METAR KJFK 020451Z 08009G17KT 7SM -RA OVC018 11/09 A3027 RMK AO2

Each part of that string has a specific meaning:

  • KJFK ICAO airport code (JFK International, New York)
  • 020451Z Observation time — day 02 of the month, 04:51 UTC
  • 08009G17KT Wind from 080° (east) at 9 knots, gusting to 17 knots
  • 7SM Visibility 7 statute miles
  • -RA Light rain ("-" = light, "RA" = rain)
  • OVC018 Overcast cloud layer at 1,800 feet
  • 11/09 Temperature 11°C / Dew point 9°C
  • A3027 Altimeter setting 30.27 inHg
  • RMK AO2 Remarks — automated station with precipitation sensor

This app fetches the latest METAR for any ICAO airport code you enter and translates it into plain English so anyone can understand the current weather at that airport.
